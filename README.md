# Get-Colleges-Name
A web application built with HTML, CSS, and JavaScript that accepts a college name as input, detects its country, dynamically changes the background image to that country, and displays a list of colleges from the same country.
This project will not function correctly on GitHub because it uses an HTTP-based API. GitHub supports only HTTPS-based APIs. To view the webpage, please download all the files and open the index.html file locally.
